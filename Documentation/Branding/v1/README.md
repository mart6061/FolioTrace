# FolioTrace Branding v1

This folder contains the first versioned brand experiment for FolioTrace.

## Files

- `foliotrace-icon.svg` - square app/icon candidate.
- `foliotrace-banner.svg` - wide banner candidate for documentation, decks, README headers, or launch screens.

## Direction

The mark combines portfolio folio layers with an event timeline ring. The palette keeps the existing trust/portfolio teal, trace/history blue, and valuation gold while moving the overall presentation closer to the current FolioTrace product name.
