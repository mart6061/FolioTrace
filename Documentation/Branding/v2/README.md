# FolioTrace Branding v2

This version provides matched light and dark SVG assets.

## Files

- `foliotrace-icon-light.svg` - light mode square icon.
- `foliotrace-icon-dark.svg` - dark mode square icon.
- `foliotrace-banner-light.svg` - light mode wide banner.
- `foliotrace-banner-dark.svg` - dark mode wide banner.

## Direction

The v2 set keeps the same folio plus event-timeline idea from v1, but splits the palette for theme-specific use:

- Light assets use a pale app-surface background, dark wordmark, and stronger teal/blue/gold strokes.
- Dark assets use a deep product background with light wordmark and softer glow details.
